### Overview
A web-based orchestration platform that manages and triggers automation workflows through an internal RPA system. Built with Firebase and Google Authentication, it provides a centralized interface for scheduling, monitoring, and tracking RPA executions across the organization.

### How It Works
Users authenticate via **Google Sign-In** to access their automation dashboard. The platform acts as an **orchestration layer** that communicates with the company's internal RPA system through API calls:

1. **Script Management**: Users organize automation workflows into logical groups within the platform
2. **Execution Triggering**: When a script is scheduled or manually triggered, the platform sends API requests to the internal RPA system
3. **Real-Time Monitoring**: The platform tracks execution status by polling the RPA system's APIs

All configuration data is stored securely in **Firebase Firestore**, with authentication handled through **Firebase Authentication** using Google OAuth. The platform serves as a user-friendly interface to manage and monitor the internal RPA infrastructure without direct script execution.

### Key Features
- **Google Authentication**: Secure, passwordless login using Google accounts
- **Script Organization**: Group related automation scripts for better management
- **Execution Dashboard**: Real-time monitoring of script runs with status tracking
- **Execution History**: Complete audit trail with detailed logs.
- **Cloud-Based**: Accessible from anywhere with automatic data synchronization

### Impact & Results
- **80% Time Savings**: Reduced manual task execution time for repetitive processes
- **Error Elimination**: Standardized execution removes human error from routine tasks
